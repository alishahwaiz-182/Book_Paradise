#  BookParadise - Online Bookstore Landing Page

A responsive online bookstore landing page built as a frontend practice project.

##  Live Demo
[Click here to view](https://alishahwaiz-182.github.io/Book_Paradise/)

##  Features
- Responsive navbar with mobile hamburger menu
- Hero banner section
- Book categories section (HTML, CSS, JavaScript)
- Featured products with star ratings and prices
- Product modals with quantity selector
- Footer with social links

##  Built With
- HTML5
- Bootstrap 5
- Font Awesome Icons
- JavaScript
